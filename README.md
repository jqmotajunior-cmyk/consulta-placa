# 🚗 Consulta de Placa

Sistema web para consulta de placas de veículos.

Projeto desenvolvido para estudo de desenvolvimento Full Stack, utilizando Node.js no backend e HTML/CSS/JavaScript no frontend.

## 🚀 Tecnologias

- Node.js
- Express
- Axios
- HTML5
- CSS3
- JavaScript

## ⚙️ Funcionalidades

✅ Consulta de placa  
✅ Comunicação com API  
✅ Interface web  
✅ Tratamento de erros  
✅ Tela de consulta com carregamento
